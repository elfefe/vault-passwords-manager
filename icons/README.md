# Icônes de l'extension

Ce dossier doit contenir les icônes suivantes :
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

Vous pouvez créer ces icônes avec n'importe quel éditeur d'images ou utiliser un générateur d'icônes en ligne.

Pour l'instant, vous pouvez utiliser des icônes temporaires ou créer des fichiers PNG simples avec un fond coloré et un symbole de cadenas.
